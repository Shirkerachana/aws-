Amazon A2I – Amazon Augmented AI
Amazon Augmented AI (A2I) is a managed service from Amazon Web Services that enables human-in-the-loop machine learning. It automatically routes uncertain or sensitive predictions from ML models to human reviewers for validation and correction.

What is Amazon A2I?
A2I removes the heavy lifting of building human review systems or managing large numbers of human reviewers. Many ML applications require humans to review low-confidence predictions — for example, extracting information from scanned mortgage forms with poor handwriting. Building such systems manually is time-consuming and expensive.

How It Works (HumanLoop)
A2I uses a HumanLoop to manage the process:

Prediction — An ML model makes a prediction.
Evaluation — A2I checks if the prediction meets a set confidence threshold.
Human Review — If confidence is low, the task is sent to a human worker via a web interface generated by A2I.
Consolidation — The human-reviewed result is sent to the output destination.


Key Features & Benefits

Built-in Workflows — Pre-built templates for document processing, image moderation, and text extraction.
Confidence Thresholds — Auto-routes low-confidence predictions to humans, reducing unnecessary manual review.
Flexible Workforce — Internal employees, third-party vendors, or Amazon Mechanical Turk.
Integration — Works with SageMaker, Rekognition, Textract, and custom models.


Use Cases

Document Processing — PII redaction, key-value extraction from forms.
Content Moderation — Reviewing images or text for inappropriate content.
Compliance & Accuracy — High-stakes decisions in finance or healthcare.
With Amazon Textract — Human review of key-value pairs in single-page documents.
With Amazon Rekognition — Human review of unsafe/low-confidence images.
With Amazon Comprehend — Review of sentiment analysis, text syntax, entity detection.
With Amazon Transcribe — Review of audio/video transcriptions to improve future results.
With Amazon Translate — Review of low-confidence translations.
Real-time ML Inferences — Review low-confidence SageMaker endpoint predictions and use results to retrain models.
Tabular Data — Integrate human review into ML apps using tabular data.
